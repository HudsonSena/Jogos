# Projeto Jogo de RPG versão 02

Jogo em linguagem C e Allegro 5.


## Stack utilizada

**Linguagem C** : Desenvolvimento de Struct, Estados, Variaveis e constantes.

**Allegro 5** : Desenvolvimento de Cores, sprites, imagens, frames, sons e outros.
